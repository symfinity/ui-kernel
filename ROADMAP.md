## Details

### 0.1.x — Foundation

Design tokens, built-in themes, Twig CSS helpers, Flex recipe, Symfony integration

### 0.2.x — DTCG and colour system

W3C design tokens, semantic colour vocabulary, OKLCH palette generation

### 0.3.x — Theme engine

Theme composition, nested themes, theme inheritance, theme overlays, runtime switching, theme capabilities

### 0.4.x — Token engine

Computed tokens, token references, token expressions, conditional tokens, token validation, token normalization

### 0.5.x — Generator engine

Palette generators, typography generators, spacing generators, radius generators, shadow generators, animation generators

### 0.6.x — Contracts

Schema versioning, theme contracts, generator contracts, export contracts, compatibility validation, extension points

### 0.7.x — Tooling

Interactive CLI, diagnostics, inspector, visual token browser, theme diffing, migration assistant

### 0.8.x — Ecosystem

Font Manager integration, Icon Manager integration, UX package integration, CMS integration, design system presets, shared manifests

### 0.9.x — Hardening

API freeze, performance tuning, documentation, cookbook, migration guides, long-term compatibility

### 1.0.x — Stable API

SemVer-stable configuration, theme contracts, token contracts, extension API, long-term compatibility
